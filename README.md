Este proyecto consiste en el desarrollo de una aplicación de consola en Java para gestionar los servicios de alquiler de vehículos de la empresa JavaDrive. El sistema permite administrar la flota de vehículos, registrar clientes, tramitar reservas y generar informes corporativos
El desarrollo se realiza de forma intermodular, aplicando conocimientos de Programación, Bases de Datos, Lenguajes de Marcas y Entornos de Desarrollo.

Siguiendo los estándares del proyecto, la aplicación se organiza en los siguientes paquetes:
model: Contiene las entidades del dominio como Vehiculo, Cliente y Reserva.
logica: Gestiona la capa de negocio, validaciones y persistencia de datos.
app: Capa de interacción con el usuario (menús y lectura de datos).
test: Repositorio de pruebas unitarias para asegurar la calidad del código.

Para cumplir con los requisitos de Entornos de Desarrollo , el equipo seguirá estas reglas:
Control de Versiones: Se utilizará Git de forma obligatoria con commits descriptivos.
Calidad: El código debe respetar la encapsulación, herencia y polimorfismo definidos en el diseño.
Documentación: Toda clase o método complejo debe estar debidamente documentado.
